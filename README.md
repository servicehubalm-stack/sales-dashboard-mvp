# MVP управленческого дашборда продаж

Готовый локальный прототип для руководителя продаж. Показывает план, факт, выполнение, прогноз, рейтинг менеджеров, лиды, КП, заявки на техническую возможность, проблемные зоны и детальную таблицу.

## Как открыть

Откройте `index.html` в браузере. Для загрузки `data.json` лучше запустить папку через локальный сервер:

```powershell
cd "C:\\Users\\1\\AppData\\Local\\Temp\\sales-dashboard-mvp"
python -m http.server 8080
```

Затем открыть `http://localhost:8080`. Если открыть файл двойным кликом, дашборд запустится на встроенных демонстрационных данных.

## Состав

- `index.html` — экран руководителя.
- `style.css` — оформление, адаптивность, подсветка проблем.
- `script.js` — загрузка данных, фильтры, KPI, графики, таблица.
- `data.json` — mock-данные.

## Логика расчетов

План считается по менеджерам, факт — по выигранным сделкам, прогноз — факт плюс 65% открытого потенциала. Проблемные зоны определяются по просроченным действиям, отсутствию следующего шага, КП без ответа более 5 дней, задержкам ТВ более 3 дней, сделкам без движения более 7 дней и задолженности.

## Ограничения MVP

Нет авторизации, промышленной БД, автоматических интеграций с CRM, биллингом, OSM и СРК. Данные демонстрационные и предназначены для согласования состава показателей и будущей промышленной реализации.
